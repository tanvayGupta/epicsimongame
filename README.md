# epicsimongame
Can you keep up? Try remembering the color sequence to set a new highscore!

## How to Play
The game will show a sequence of colors by highlighting the buttons and playing a sound. Your goal is to repeat this sequence by clicking the buttons in the exact same order.

*   If you correctly replicate the sequence, the game will add a new color to the end of the sequence, making it one step longer.
*   If you click a button out of sequence, the game is over.
*   After a game over, you can press any key to restart the game and try for a new highscore.

## Technologies Used
*   HTML
*   CSS
*   JavaScript
*   jQuery

## Running Locally
To run this game locally on your computer:
1.  Download all the project files.
2.  Ensure you maintain the directory structure as described below.
3.  Open the `index.html` file in your preferred web browser.

## Project Structure
The project files are organized as follows:

*   `index.html`: The main HTML file that you open to play the game.
*   `css/styles.css`: Contains all the styling rules for the game's appearance.
*   `js/game.js`: Holds the core JavaScript logic that powers the game mechanics.
*   `assets/sounds/`: This directory contains all sound files used for game events (button presses, wrong moves).
    *   `blue.mp3`
    *   `green.mp3`
    *   `red.mp3`
    *   `yellow.mp3`
    *   `wrong.mp3`
*   `README.md`: This file, providing information about the project.
